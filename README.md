# Scheduler

## About

A simple action scheduler

## Setup
To start the project will be necessary run the following scripts:
```bash
#Clone the repository
$ git clone https://github.com/ravinmor/scheduler

#Access the backend app folder
$ cd scheduler

#Install packages
$ yarn

#Start the app
$ yarn start

```
